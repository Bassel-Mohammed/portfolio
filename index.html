Snake Game

The snack game is a simple and interactive application built using Java and the Swing GUI framework. The player controls a snack, which moves around the screen collecting food while avoiding obstacles. The game starts with a basic interface and uses key events for player movement. The goal is to collect as many food items as possible without hitting the walls or the player's own tail. Each food item increases the player's score and the length of the snack, making the game progressively more challenging. The game provides real-time feedback on the score and supports a restart feature when the player loses.


Technologies Used: Java , Java Swing

Desktop Calculator

a basic desktop application created using JavaFX for the graphical user interface. Technologies Used: Java , JavaFx.
